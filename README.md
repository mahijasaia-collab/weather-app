# 🌤 Advanced Weather App

🔗 Live Demo: https://mahijasai-collab.github.io/weather-app/

---

## 📌 Project Overview

This is a modern weather application that provides real-time weather updates, including temperature, conditions, a 5-day forecast, and location-based updates with a dynamic UI.

---

## 🚀 Features

* 🌍 Auto-detect user location (Geolocation API)
* 🔍 Search weather by city name
* 🌡 Real-time temperature & weather conditions
* 🤔 Feels-like temperature
* 💧 Humidity & wind speed
* 🌅 Sunrise & 🌇 Sunset timings
* 📊 5-Day Weather Forecast
* 🌞🌙 Dynamic Day/Night UI
* 🎨 Background changes based on weather
* 💎 Glassmorphism UI design

---

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* OpenWeather API

---

## ⚙️ How to Run Locally

1. Clone the repository  
2. Open the folder  
3. Open `index.html` in your browser
